# Play Store Listing Draft

## App Name

Muslim Time

## Short Description

Namaz vaxtları, Quran, qibləgah, bildirişlər və gündəlik dini köməkçi.

## Full Description

Muslim Time gündəlik ibadət həyatını rahatlaşdırmaq üçün hazırlanmış dini tətbiqdir.

Tətbiq daxilində bunlar mövcuddur:

- namaz vaxtları və məkan seçimi
- avtomatik və manual məkan təyini
- namaz xatırlatmaları və öncədən bildirişlər
- cümə və ad günü üçün xüsusi mesajlar
- Quran kitabını oxu
- Quran dinlə
- Allahın adları
- qibləgah
- çoxdilli dəstək
- açıq, tünd və sistem teması

Namaz vaxtı mənbəyi və məkan seçimləri istifadəçinin bölgəsinə uyğun qurula bilir. Tətbiq həm sadə istifadə, həm də gündəlik rahatlıq üçün hazırlanıb.

Quran bölməsində üç ayrıca giriş var:

- Quran kitabını oxu
- Quran dinlə
- Allahın adları

Bu yanaşma tətbiqin daha rahat və daha anlaşılan istifadəsini təmin edir.

Qibləgah bölməsində kompas və xəritə görünüşü mövcuddur. İstiqamət və kalibrasiya statusu daha aydın formada göstərilir.

Bildiriş sistemi daxilində:

- namaz vaxtında bildiriş
- namazdan əvvəl xatırlatma
- təkrar xatırlatma
- cümə bildirişi
- ad günü bildirişi

Muslim Time gündəlik istifadə üçün sadə, aydın və dini məqsədə uyğun bir təcrübə təqdim edir.
